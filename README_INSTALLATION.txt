INSTALLATION SUR GITHUB PAGES

1. Va sur github.com et connecte-toi.
2. Clique sur New repository.
3. Nom du dépôt conseillé : info-demon
4. Coche Public puis Create repository.
5. Envoie tous les fichiers de ce dossier : index.html, style.css, script.js et le dossier assets.
6. Va dans Settings > Pages.
7. Dans Branch, choisis main puis /root, puis Save.
8. Attends 1 à 3 minutes.
9. Ton site sera disponible à une adresse du type : https://tonpseudo.github.io/info-demon/

MODIFICATION DES DÉMONS
Ouvre script.js puis modifie la liste "demons".

COULEURS
Ouvre style.css puis modifie les valeurs dans :root.
