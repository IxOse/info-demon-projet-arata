INSTALLATION SIMPLE

1. Supprime tout dans ton dépôt sauf README.md si tu veux.
2. Extrais ce ZIP.
3. Envoie directement à la racine :
   - index.html
   - style.css
   - script.js
   - assets
4. N'envoie PAS le fichier ZIP sur GitHub.
5. Settings > Pages > main > /(root).
