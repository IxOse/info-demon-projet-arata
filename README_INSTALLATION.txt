INSTALLATION GITHUB PAGES

1. Envoie tous ces fichiers à la racine du dépôt :
   - index.html
   - style.css
   - script.js
   - assets/

2. Va dans Settings > Pages.
3. Source : Deploy from a branch.
4. Branch : main.
5. Folder : / (root).
6. Save.
